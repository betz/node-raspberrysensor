# raspberrysensor

Library for accessing various sensors via the bcm2835 library.

## Getting Started
Install the module with: `npm install raspberrysensor`

```javascript
var raspberrysensor = require('raspberrysensor');
raspberrysensor.awesome(); // "awesome"
```
